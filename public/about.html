How i will build your website ?

  1. First phase ( design ) : i will study your project carefully and then i will make a mockup for the project and then present it to you, if you don’t like it, then i will make another one for you, if you like it, then i will immediately start working on the next phase.
  2. second phase ( coding - front-end ) : in this phase i will start to code the design made in the last phase, and in this phase i use tools and coding languages like html & css & javascript & bootstrap & css tailwind & React.js, these tools and languages are responsible of building & styling the Front-End of your website.
  3. Third phase ( coding - backend) : coding the Front-end of your website is over, then it’s time to code the Back-End, Back-end is a term refers to backstaging of your website like linking your website front-End with databases, setting up a server for the website using express.js and linking website pages using RestAPI and to set all this technologies we need an environment, in our case i will use Node.js and the reason we use Node.js is most of the libraries we use is from javascript like React.js so Node will achieve sort of flexibility & harmony in your website, in case we need to deal with users data in your website we will use security methods like salting & hashing in your website and this help us to encrypt users sensitive data like passwords, email adress and card details. the last thing we will do in this phase is setting database and the most preferred way for me is to use MongoDB because it’s safe, reliable, and trusted by big companies to run mission critical databases.
  4. phase number 4 ( Testing ): after coding the entire website, there’s just one phase stop us from launching it and this phase is Testing, at the end we don’t want a website with hundreds of mistakes, and in this phase i use tool called cypress to asure that your website will work flawless, fast, secure and functioning.
  5. last phase ( Hosting ) : after substantial amount of milestones, we have reached the last phase, hosting you website, in this phase your website will be online, you can name your website whatever you want, and if this name (domain) is available, i will be able to book it and install it to your website, plus, i will install ssl certificate to make your website url start with (https) and this is important when it comes to users trust, this phase will be accomplished using Heroku.
  6. post hosting phase : now you’re all set, your website is online on the internet, there is just few things you need, the first is you will recieve your source code documented in case you have another plan for maintaining your website, the second is if you don’t have a plan for maintaining your website, don’t worry, i’m here to help you, all you need to do is infrom me if you want me to be responsible for this task.
